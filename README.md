# GraphicsComputation3D_Java

Game developed for Graphics Computation course using Java and Java3D API.

## Technologies
- Java
- Java3D API
- Eclipse IDE

## Features
- 3D graphics rendering
- Camera controls
- Object transformations
- Lighting system
- 3D collision detection

## Setup
1. Install Java + Java3D
2. Clone repository 
3. Open in Eclipse
4. Run Main.java
